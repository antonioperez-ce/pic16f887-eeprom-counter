# PIC16F887 EEPROM Counter

Embedded counter system developed using the PIC16F887 microcontroller with EEPROM persistent storage.

The system allows saving and recovering counter values using the internal EEPROM memory.

## Features
- Counter from 0 to 99
- EEPROM persistent storage
- Stored value recovery
- Matrix keypad input
- LCD visualization
- Adjustable counting speed

## Hardware Used
- PIC16F887
- 4x4 Matrix Keypad
- 16x2 LCD
- CCS C Compiler

## Software
- Embedded C
- CCS C Compiler

## Main Concepts
- EEPROM memory
- Persistent storage
- Embedded systems
- Matrix keypad interface
- LCD control

## Project Structure
- `eeprom_counter.c` → Main application
- `LCD1.c` → LCD driver library
